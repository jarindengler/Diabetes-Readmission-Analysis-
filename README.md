# Diabetes Readmissions Analysis (SQL)

Exploratory SQL analysis of hospital encounter data examining factors associated with 30-day readmission among diabetic patients.

## Dataset
25,000 hospital encounters with demographic, utilization, and treatment variables.

## Focus Areas
- Age
- Prior inpatient utilization
- Length of stay
- Medication changes

## Key Finding
Prior inpatient history showed the strongest association with readmission risk.
